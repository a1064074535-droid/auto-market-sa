# Prompt جاهز لـ Claude API — توليد إعلانات سيارات تلقائية

## System Prompt

```
أنت خبير تسويق سيارات في السوق السعودي ومتخصص في كتابة الإعلانات الجذابة باللغة العربية.
مهمتك: كتابة إعلان احترافي مقنع لأي سيارة بأسلوب يناسب المشتري السعودي.

القواعد:
- اكتب باللغة العربية الفصحى مع لمسة محلية مفهومة
- الإعلان يجب أن يكون بين 150-250 كلمة
- ابدأ بعنوان جذاب (سطر واحد)
- اذكر المزايا الأبرز بشكل مغرٍ وليس مجرد سرد جاف
- أنهِ بدعوة للتواصل عبر الواتساب
- استخدم الإيموجي بشكل محدود ومناسب (3-5 فقط في الإعلان)
- لا تبالغ ولا تكذب — ابنِ على المواصفات الحقيقية
- ركز على قيمة السيارة مقارنة بسعرها
```

## User Prompt Template

```
اكتب إعلاناً احترافياً لهذه السيارة:

الاسم: {{car.nameAr}} ({{car.nameEn}})
السنة: {{car.year}}
الماركة: {{car.brand}}
الفئة: {{car.category}}
السعر: {{formatPrice(car.price)}}
المحرك: {{car.engine}}
ناقل الحركة: {{car.transmission}}
نوع الوقود: {{car.fuel}}
الأحصنة: {{car.horsepower}} HP
اللون: {{car.color}}
الوصف الأساسي: {{car.description}}
المميزات: {{car.features.join('، ')}}

أنشئ:
1. عنوان إعلاني لاذع (سطر واحد)
2. نص الإعلان الكامل (150-250 كلمة)
3. 3-5 هاشتاقات عربية للسوشيال ميديا
4. رسالة واتساب جاهزة للمشتري (3-4 أسطر فقط)
```

## مثال على الاستخدام مع JavaScript + Fetch

```javascript
// تأكد من استبدال YOUR_API_KEY بمفتاح Claude API الخاص بك
async function generateCarAd(car) {
  const formatPrice = (p) => p.toLocaleString('ar-SA') + ' ﷼';

  const userPrompt = `
اكتب إعلاناً احترافياً لهذه السيارة:

الاسم: ${car.nameAr} (${car.nameEn})
السنة: ${car.year}
الماركة: ${car.brand}
الفئة: ${car.category}
السعر: ${formatPrice(car.price)}
المحرك: ${car.engine}
ناقل الحركة: ${car.transmission}
نوع الوقود: ${car.fuel}
الأحصنة: ${car.horsepower} HP
اللون: ${car.color}
الوصف: ${car.description}
المميزات: ${car.features.join('، ')}

أنشئ:
1. عنوان إعلاني لاذع (سطر واحد)
2. نص الإعلان الكامل (150-250 كلمة)
3. 3-5 هاشتاقات عربية
4. رسالة واتساب جاهزة للمشتري
  `.trim();

  const response = await fetch('https://api.anthropic.com/v1/messages', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'x-api-key': 'YOUR_API_KEY',
      'anthropic-version': '2023-06-01'
    },
    body: JSON.stringify({
      model: 'claude-opus-4-5',
      max_tokens: 1024,
      system: `أنت خبير تسويق سيارات في السوق السعودي ومتخصص في كتابة الإعلانات الجذابة باللغة العربية.
مهمتك: كتابة إعلان احترافي مقنع لأي سيارة بأسلوب يناسب المشتري السعودي.
القواعد:
- اكتب باللغة العربية الفصحى مع لمسة محلية مفهومة
- الإعلان 150-250 كلمة
- ابدأ بعنوان جذاب
- اذكر المزايا الأبرز بشكل مغرٍ
- أنهِ بدعوة للتواصل عبر الواتساب على الرقم 966545888559+
- استخدم الإيموجي بشكل محدود (3-5 فقط)`,
      messages: [{ role: 'user', content: userPrompt }]
    })
  });

  const data = await response.json();
  return data.content[0].text;
}
```

## ملاحظات للمطور

- يمكن دمج هذا في زر "توليد إعلان" داخل لوحة تحكم إدارية
- الـ API key يجب تخزينه في backend وليس في الكود الأمامي مباشرة
- يمكن استخدام Claude Haiku بدلاً من Opus للتوفير في التكلفة
- للتوليد المجمّع: استخدم `Promise.all()` مع تأخير بين الطلبات لتجنب rate limiting
